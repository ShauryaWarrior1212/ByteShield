Cybersecurity 101

Introduction

Cybersecurity involves protecting computer systems and networks from information disclosure, theft of or damage to their hardware, software, or electronic data. It's crucial in today's digital age.

Common Threats

  • Malware
  • Phishing
  • Man-in-the-Middle Attacks
  • Denial-of-Service Attacks

Sample JavaScript Function for Input Validation


// JavaScript Function to Validate Email
function validateEmail(email) {
  const re = /^(([^<>()\[\]\\.,;:\s@"]+(.[^<>()\[\]\\.,;:\s@"]+)*)|
  (".+"))@(([^<>()[\]\\.,;:\s@"]+.)+[^<>()[\]\\.,;:\s@"]{2,})$/i;
  return re.test(String(email).toLowerCase());
}

console.log(validateEmail("test@example.com")); // true

Sample Python Script for Password Strength Check


# Python Function to Check Password Strength
import re

def check_password_strength(password):
    if len(password) < 8:
        return "Weak Password: Less than 8 characters."
    if not re.search("[a-z]", password):
        return "Weak Password: No lowercase letters."
    if not re.search("[A-Z]", password):
        return "Weak Password: No uppercase letters."
    if not re.search("[0-9]", password):
        return "Weak Password: No numbers."
    if not re.search("[_@$]", password):
        return "Weak Password: No special characters."
    return "Strong Password!"

print(check_password_strength("P@ssw0rd"))

Best Practices

Always keep your software updated, use strong and unique passwords, be wary of suspicious emails, and consider using two-factor authentication wherever possible.
